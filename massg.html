MASSG

MULTIVERSE AUTOMATED STORM-SUPPRESSION GRID — ADD-ON & JAVA MOD

MASSG is a Minecraft Add-On (Bedrock Edition) and Java Mod created by Logan Wall (@REWRITTENTM). It brings the in-universe MASSG Operating System directly into Minecraft — custom mobs, mechanics, items, and lore tied to every block of the Devouring Storms universe.

The mod isn't just a gameplay expansion — it's a narrative artifact. Installing it makes you a participant in the story. Every play session is canon. The MASSG entity watches.

Available for Minecraft Bedrock Edition (Add-On) and Java Edition (Mod). Both in active development.

▲ MASSG ADD-ON / JAVA MOD — OFFICIAL TRAILER

WHAT'S IN THE MOD

🌀

DIMENSIONAL MECHANICS

Custom dimensions tied to the Devouring Storms lore. Each dimension represents a timeline in the Rewritten Multiverse — some stable, some already consumed by the storm.

👾

MASSG ENTITIES

Custom mobs that embody the storm's influence. They behave differently depending on timeline stability. Some are enemies — some are echoes of characters from the series.

🔧

OS INTERFACE ITEMS

In-game items that mirror the MASSG OS interface. Terminals, data drives, access cards — every item has a lore entry that deepens the story.

📖

FOURTH-WALL LORE

The mod knows you're playing it. MASSG monitors every action. Lore entries react to what you do in-game. The line between player and character is intentionally blurred.

☕

JAVA EDITION

Full Java Mod port — same experience, deeper integration with Java Edition's modding capabilities. Both versions are developed in parallel by Logan Wall.

📱

BEDROCK ADD-ON

Full Bedrock Add-On — works on all platforms that run Bedrock Edition. Cross-platform compatible. Drop it into your world and the storm follows you in.
